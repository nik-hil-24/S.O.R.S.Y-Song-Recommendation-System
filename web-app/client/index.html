About The Project


A Song Recommendation System was created with Python, designed to generate tailored song suggestions based on an inputted playlist ID. Through the use of the Spotipy API, key song data was extracted, cleaned, and visualized to enhance the analytical process. The application of sentiment analysis and collaborative filtering techniques further enabled the generation of personalized song recommendations. The final product also included a user-friendly web interface, developed using HTML, CSS, JS, and Flask, for a smooth user experience.
